Join the ROS Agriculture community

ROS Agriculture is an open source project that anyone in the community can use, improve, and enjoy. We'd love you to join us! Here's a few ways to find out what's happening and get involved.

Learn and Connect

Using or want to use ROS Agriculture? Find out more here:

  • Twitter: Follow us on Twitter to get the latest news
  • Facebook: Follow us on Facebook to get the latest stories
  • Slack: Discussion and help from your fellow users

Develop and Contribute

If you want to get more involved by contributing to ROS Agriculture, join us here:

  • Slack: Chat with other project contributors
  • GitHub: Development takes place here!

You can find out how to contribute to these docs in our Contribution Guidelines.

Core Values

  • Expedition Behavior - We take calculated risks. Embrace failure.
